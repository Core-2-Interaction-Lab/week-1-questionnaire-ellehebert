In the body the site begins with a navigational feature that includes the company's logo on the left and a hamburger menu bar on the right. The "hamburger-menu" bar allows the user to navigate through the sites pages and is sectioned class under the "navbar" class.


Within the "navbar" there is another class, "section-top-lines" that serves as an aesthetic funtion. They were built with a flexbox that allows them to condense or widen depending on the users screen size.

The company's logo, class="raca-logo-lines" is placed below the navigation bar. It has a color overlay effect when cursor hovers over the text.
